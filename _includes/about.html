Training

I have been involved in planning, creating and delivering a number of training courses and programmes.

  • The NCAS Introduction to Scientific Computing Course

    Getting started with Python

    I have over 8 years of experience in delivering Python training as part of the NCAS ISC course. This involves a beginners' guide to Python programming followed by tools and libraries for working with scientific data.

  • The JASMIN Workshop

    Introducing users to the JASMIN Platform

    I have co-developed the JASMIN Workshop, a set of interactive exercises to guide users on how to access and use our data and compute platform.

  • JASMIN Webinars

    Web-based teaching and information sharing

    As part of the JASMIN Webinar series, I have written and presented short sessions on a range of topics. These include (1) Git and GitHub, (2) the JASMIN Jupyter Notebook Service and (3) tools for running efficient workflows.

  • In-house training

    Supporting colleagues

    In my current role, I enjoy sharing best practice, tips and tricks with colleagues. These range from informal chats to organised sessions across a range of topics, such as: Linux shell, Python, Git/GitHub, Unit testing, Continuous integration, Django basics, Ansible usage, working with data, environmental data tools.
